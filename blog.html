What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree of objects, where each node corresponds to a part of the document, such as elements, attributes, and text. The DOM allows programs (like JavaScript) to access and manipulate the document's structure, content, and style dynamically, enabling changes to be made to a web page without reloading it.

How do you select an element from the DOM?

To select an element from the DOM using JavaScript, you can use several methods provided by the document object. Here are the most common ones:

  • getElementById()
  • getElementsByClassName()
  • getElementsByTagName()
  • querySelector()
  • querySelectorAll()
    • What is event delegation in the context of the DOM, and why is it useful?

    Event delegation is a technique in the DOM where instead of attaching event listeners to individual child elements, you attach a single event listener to a parent element. The parent element listens for events that bubble up from its child elements, allowing you to handle events more efficiently, especially in cases where child elements are dynamically added or removed.
